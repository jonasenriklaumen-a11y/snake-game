# SnakeGame 🐍

Ein kleines klassisches Snake-Spiel in Python, gebaut mit **Tkinter** (Standardbibliothek — keine Installation von Zusatzpaketen nötig).

## Spielen

Voraussetzung: Python 3.8+ (Tkinter ist bei den meisten Python-Installationen bereits enthalten).

```powershell
git clone https://github.com/<dein-benutzername>/SnakeGame.git
cd SnakeGame
python snake.py
```

## Steuerung

| Taste | Aktion |
|---|---|
| Pfeiltasten | Richtung ändern |
| Leertaste | Nach „Game Over“ neu starten |

Ziel: Iss die roten Punkte, ohne gegen die Wand oder dich selbst zu laufen. Jeder Punkt gibt +1 und lässt die Schlange wachsen.
